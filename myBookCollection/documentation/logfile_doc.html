Log File Documentation


Log File is only for Admin to keep track of the activities that is being done in the system. In order to view the log file, the user will need to login with proper admin credetials. Normal Users are not allowed to view the log file. Below is the picture of Log File:

Log File Represent:


  1. Admin will be able to see the chnages done in the system.

    Admin can see:

    • What was added.

    • What was edited.

    • What was removed.

  2. Admin will also keep track of the date and time the changes happened.

  3. The Admin will see the latest Log File used by the system.
