Welcome to iBooks Documentation


The iBooks Documentation is the collection of documents that describes the requirements, capabilities, limitations, design, operation, and maintenance of a system, such as a communications, computing, or information processing system. The documentation is organized in in such a way that when you open the View Collection tab, it will show the documentation of the View Collection tab which is in the main page iBooks Collection and same for others.



Navbar or Navigation Bar


The navigation bar on top on every page will help the user to access different pages and perform different task. You can see the navbar below:



Some of the task that user will be able to perform are:


  1. iBooks Collection - Home page of the system

  2. View Collection - viewing the collection

  3. Add Collection - adding new books to the existing collection

  4. Conditional Delete - deleting some information about the books from the existing collections

  5. Delete All - removing books from the existing collections

  6. Edit Collection - editing the information of perticular book

  7. XML File - viewing the XML (database) file online

  8. iBooks Documentation - read the documentation of the system. Learn how his system was created, who created and the most important how to use it.
