Add Collection Documentation


Add Collection tab will display a form that will allow users to input the information about the books accordingly. The form will ask everything that is needed to add into the system. Below is the picture of Add Collection tab:



Form requirements and action:

  1. The Book ID represent the uniqeness of a book. As we have many books, it will be difficult to search for books with the its name. As some books might have same name or even same author. So to avoid that clash, the user will have to create a unique Book ID which will be later linked with other information.
    *Note: User will have to add a unique ID otherwise the system will not allow to add anything. Example: BK9008, PT877, etc.

  2. The Author represent the author or the person who wrote that specific book. So the user will add the author name that correspond with the book name. Example: Nelson Mandela, William Shakespear, etc.

  3. The Title represent the title of the book that the specific author wrote. So the user will have to add the name of the book which is written by the author. Example: Midnight Rain, A Tale of Two Cities, etc.

  4. The Genre represent the genre of the book. So the user will add the genre of the book they desire to add into the system. Example: Science, History, Politics, Maths, etc.

  5. The Price represent the price of the books in dollars. So the user will add the price of the book they desire to add into the system. Example: $10.99, $299.25, etc.

  6. The Published Date represent the date that the book was published on. So the user will add the publishing date of the book that they desire to add. Example: 12/2/1998, 22/3/2008, etc.

  7. The Description represent the summary or the discription of the book. So the user will add the summary of the book which will describe what the book is about. Example: A former architect battles corporate zombies, an evil sorceress, and her own childhood to become queen of the world.
