About Us



This system is build in order to maintain employee's data is softcopy or databse. The user can only view the current employees, on going projects and existing departments. The admin or the head of deaprtments can add employees, add departments and add projects. What ever the user type in the input box, the box will then automatically add the inputted text into the sql query and execute it. The URL to the application is http://natto.mooo.com/~db4140669/ems/. The application is username/password protected. This application contain 7 screens which include EM Management screen (home screen), Employee screen, Project screen, and Department screen.


The Main Screen and Employee Management Screen



The main functionality presented on this screen are:


(1) The main menu allow users to see where they are and who they are logged in as such as (Admin or Department Head). User can also select the Employee Information Menu to directly gain access to the database view of the employee's information.

(2) This fucntion allow user to input information about the current employees and extract the information as they desired.This fucntion uses SELECT sql command. For instance, user want to search the employee with the salary of $5000 then user will have to enter 5000 in the salary box. Moreover, they can also reset/clear all by just pressing "Reset" button. If user input invalid data/empty the alert box will apear saying please re-enter.

(3) This fucnion will allow user to insert new employees or delete existing employee. This fucntion uses INSERT and DELETE sql fucntions in order to add and remove the employee. If user input invalid data the alert box will apear saying please re-enter.
