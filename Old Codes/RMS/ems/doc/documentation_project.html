Project's User Documentation


This fucntion allow user to view the existing projects in each departments, it allows authorized user to add new projects to any department, moreover it also allow authorized users to assign any employee to any existing projects.


The Project Screen





The main functionality presented on this screen are:

  1. The navbar allow users to see the name of the applcation they using, the employee tab, add employee tab and documentation tab.

    • EM Management Tab

         The EM Management Tab is a linked tab which will take the user to the home page of the application where can access all the fucntions in this    application.

    • Project Tab

         The Project Tab will allow user to see all the existing projects in a department.

    • Add Project Tab

         The Add Project Tab is a linked tab which will take the user to a page where the user can add new projects to any departments.

    • Assign Project Tab

         The Assign Project Tab is a linked tab which will take the user to a page where the user can assign any existing projects to any employee    working in any department.

    • Documentation Tab

         The Documentation Tab is a linked tab which will take the user to the documentation section. Where user can see the description of different    fucntions of the system.

  2. This table allow users to view the current projects existed with project number, project location and the department where the project is for.

Add Project Screen





The main functionality presented on this screen are:

  1. This form allow users to add new employees in an organization. This fucntion are only for authorized people like admin and department heads. The user can add SSN, Name, Salary, Department they work in, and so on.

    • Submit Button

         The submit button will submit the form and update the database with the newly added employee.
