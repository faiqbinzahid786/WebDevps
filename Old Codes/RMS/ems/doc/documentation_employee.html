Employee's User Documentation


This fucntion allow user to view the existing employees in the organization, moreover it also allow authorized user to add new employees to the organization. Authorized people can also assign different projects to different employees anytime.


The Employee Screen





The main functionality presented on this screen are:

  1. The navbar allow users to see the name of the applcation they using, the employee tab, add employee tab and documentation tab.

    • EM Management Tab

         The EM Management Tab is a linked tab which will take the user to the home page of the application where can access all the fucntions in this    application.

    • Employee Tab

         The Employee Tab will allow user to see all the existing employees in an organization.

    • Add Employee Tab

         The Add Employee Tab is a linked tab which will take the user to a page where the user can add new employees to the organizational system.

    • Documentation Tab

         The Documentation Tab is a linked tab which will take the user to the documentation section. Where user can see the description of different    fucntions of the system.

  2. This table allow users to view the current employees existed in an organization. The user can view every employee's data (ssn, name, salary, department they work in, or works on).

    • Projects Button

         The projects button takes user to the assign project screen where the user can assign any project for any departments.


Add Employee Screen





The main functionality presented on this screen are:

  1. This form allow users to add new employees in an organization. This fucntion are only for authorized people like admin and department heads. The user can add SSN, Name, Salary, Department they work in, and so on.

    • Submit Button

         The submit button will submit the form and update the database with the newly added employee.
