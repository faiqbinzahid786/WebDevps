Welcome to The Documentation EMS Management


This system is build in order to maintain employee's data is softcopy or database. The user can only view the current employees, on going projects and existing departments. The admin or the head of deaprtments can add employees, add departments and add projects. There is a form in each category(employee, project and department) which allow only the admin to insert new inputs. The URL to the application is http://natto.mooo.com/~db4140669/ems/. The application is username/password protected which means only authorized user can see the informations. This application contain 11 screens which include EM Management screen (Home screen), Employee screen, Project screen, Department screen and Documentation.
